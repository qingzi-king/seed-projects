## 项目运行
```

这是一个基于react-v16.13.1、Antd-v4.4.3、craco-v5.6.4进行封装的一个骨架项目（内容为示例，非真实数据）。

```

## 项目运行

```
cnpm i # 安装依赖（或npm i或yarn）
npm start # 本地启动

```

## 目录结构

```
├── LICENSE

├── craco.config.js      # craco配置
├── package.json
├── public
│   ├── favicon.ico
│   ├── index.html        # 页面模板
│   ├── manifest.json
│   └── robots.txt
├── src
│   ├── App.js
│   ├── assets            # 资源目录
│   ├── components        # 公共组件目录
│   ├── index.js          # 入口文件
│   ├── pages             # 页面目录
│   ├── serviceWorker.js
│   ├── services          # 服务目录
│   ├── setupTests.js
│   └── store             # 数据状态目录
└── yarn.lock
```
